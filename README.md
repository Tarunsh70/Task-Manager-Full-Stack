# Task Manager Full Stack Application

A modern full-stack task management web application built using the MERN Stack.

## 🚀 Live Links

Frontend:
https://task-manager-full-stack-production-7e5b.up.railway.app

Backend:

---

## ✨ Features

- User Authentication
- Project CRUD
- Task CRUD
- Dashboard Statistics
- Protected Routes
- Responsive UI

---

## 🛠️ Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Axios
- Vite

Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication

Deployment:
- Vercel
- Railway

---

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Tarunsh70/Task-Manager-Full-Stack.git
```

### Backend Setup

```bash
cd server
npm install
npm run server
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```
